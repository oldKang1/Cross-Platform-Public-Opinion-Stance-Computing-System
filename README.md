XHS Opinion
小红书舆情立场与情绪演化分析系统

一个面向小红书评论的 多维舆情分析与可视化平台，
融合 自然语言处理（NLP）+ 数据分析 + 可视化系统，
用于对网络舆论中的 立场、情绪、地区分布与时间演化 进行建模与展示。

1. 项目背景

社交媒体已成为公共舆论的主要载体。
小红书等平台上的用户评论包含了大量 观点冲突、情绪波动与群体分化信息，
但这些信息以非结构化文本存在，难以直接分析。

本项目的目标是将评论从：

“一堆文本” → “可量化、可比较、可解释的舆情结构”

即：
把评论转化为：立场 × 情绪 × 时间 × 地区 的多维数据空间

2. 系统功能

系统支持从评论 CSV 数据出发，自动完成以下流程：

（1）文本理解

立场识别：支持 / 反对 / 中立

情绪分析：正面 / 中性 / 负面

输出每条评论的分类结果与置信度

（2）统计分析

情绪分布

立场分布

地区分布（IP属地）

时间演化（按天 / 时间片）

（3）可视化

地区情绪堆叠柱图

情绪分布饼图 / 柱图

立场结构

进度条 + 数值看板

动态 Web 仪表盘（show 页面）

3. 技术路线
CSV 评论数据
     ↓
文本清洗（去空 / 统一编码）
     ↓
NLP模型推理
  ├── 情绪分类
  └── 立场分类
     ↓
结果表（含置信度）
     ↓
spark-Hive
     ↓
Matplotlib / 图表生成
     ↓
Flask Web 可视化

4. 使用的核心技术

前端	HTML · TailwindCSS · Three.js

5. 数据字段

输入 CSV 至少包含：

字段	含义
content	评论文本
ip	评论 IP 属地
time	发布时间

系统会自动生成：

新字段	含义
sentiment	情绪类别
sentiment_score	情绪置信度
stance	立场类别
stance_score	立场置信度
